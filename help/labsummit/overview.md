# 実習の概要と概要

Document Cloudとその中核製品であるAcrobatとSignが、現在最も広く使用されている生産性ツールの一部であることをご存知でしたか。 Fortune 100社の80%以上の企業で、ビジネス・チームとマーケティング・チームが使用しています。

このラボでは、次の方法を学び、生産性と創造性を次のレベルに引き上げます。

* PDFでデジタル化
* PDFでさらに作業を行う
* 顧客と従業員の経験を変換
