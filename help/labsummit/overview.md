# 実習の概要と概要

Document Cloudとそのコア製品であるAcrobatとSignは、現在最も広く使用されている生産性向上ツールの一部であることを知っていましたか？ フォーチュン100社の80%以上の企業でビジネスおよびマーケティングチームが使用しています。

このラボでは、次の方法を学習して、生産性と創造性を次のレベルに引き上げます。

* PDFでデジタル化
* PDFをさらに活用
* 顧客エクスペリエンスと従業員エクスペリエンスの変換
