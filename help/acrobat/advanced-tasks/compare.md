# 2つのPDF間の相違を検出する

間違ったバージョンのファイルを使って作業するという誤った操作は決してしないでください。 2つのPDFファイルの違いを迅速かつ正確に検出し、ドキュメントレビューのワークフローを改善します。

>[!NOTE]
>
>Acrobat Pro DCでのみ使用可能です。
