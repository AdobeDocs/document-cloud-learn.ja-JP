# カスタムコマンドとツール

カスタムコマンドとカスタムツールを使用して、文書ワークフローの効率を向上させる方法について説明します。 次に、新しいコマンドやツールを同僚と共有して、組織の効率を向上させます。 このビデオチュートリアルは、[新しいAcrobatエクスペリエンス](../getting-started/new-experience.md)ではまだ有効になっていません。

>[!NOTE]
>
>カスタムコマンドは、Acrobat Pro、プレミアムまたはStudioでのみ使用できます。
