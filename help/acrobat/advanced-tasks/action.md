# ガイド付きアクション

最高のカスタマーエクスペリエンスを提供するには、これまでになく高速にコンテンツを作成する必要があります。 ガイド付きアクションを使用して、ワンクリックで多数のドキュメントにわたって1つ以上のコマンドを実行することで、時間とキーストロークを節約する方法を説明します。 このビデオチュートリアルでは、[新しいAcrobatエクスペリエンス](../getting-started/new-experience.md)を使用します。

>[!NOTE]
>
>Acrobat Pro、プレミアム、またはStudioのみで使用可能です。
