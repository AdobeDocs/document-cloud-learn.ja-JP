# 60秒のAcrobatの概要

60秒のAcrobatには、Acrobatで新しいトリックを1分以内に学ぶのに役立つ一口サイズのチュートリアルが用意されています。 これらのタスクベースのヒントは、Acrobatの隠れた機能のロックを解除することで、PDFファイルを操作するための新しいスキルを身に付けるのに役立ちます。 回答を素早く得るには1つ、文書の生産性を向上させるには5つ見ることができます。それでも、休憩を楽しむ時間はあります。
