# ユースケースの概要

Acrobatを使用して生産性を向上させ、情報をチームや業界向けの実用的なインサイトに変換する方法を説明します。

## 基幹業務

異なる基幹業務全体を担当するチームがAcrobatを使用して、日常的な文書業務の課題の解決、ワークフローの合理化、およびビジネスクリティカルな作業のサポートを行う方法について説明します。

## 新機能

>[!BEGINTABS]

>[!TAB ボトルネックのないITの有効化]

[ITチーム](lob/it/it-enablement.md)がAcrobat Studioを使用して、文書ワークフローを合理化し、セキュリティとコンプライアンスを強化し、組織全体にガバナンスプログラムを拡張する方法を説明します

>[!TAB Acrobat Studioを使用した製品発売の促進]

マーケティングチームがAcrobat Studioを使用して、1つのPDFスペースから[製品発表](lob/marketing/marketing-product-launch.md)のアセットを一元化し、関係者のレビューを効率化し、組織の対応状況を調整する方法について説明します。

>[!ENDTABS]
