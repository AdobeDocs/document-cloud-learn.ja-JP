# Acrobat ITユースケースの概要

ITチームがAcrobat Studioを使用して、文書ワークフローを合理化し、セキュリティとコンプライアンスを強化し、組織全体にガバナンスプログラムを拡張する方法について説明します。

## 新機能

>[!BEGINTABS]

>[!TAB ボトルネックのないITの有効化]

[ITチームがAcrobat Studio](it-enablement.md)を使用して、ワークロードを増やしたり制御を犠牲にしたりすることなく、機密性の高いドキュメントの保護、ベンダーとの契約の分析、大規模なガバナンスポリシーの展開を行う方法について説明します。

>[!ENDTABS]
