# Acrobat financeユースケースの概要

Acrobatが、レポート作成、予算作成、監査プロセス全体で、文書作成、データ分析、承認、セキュリティ、コンプライアンスをどのようにサポートしているかを示す、財務固有のワークフローについて説明します。

## 新機能

>[!BEGINTABS]

>[!TAB 財務レポートを経営陣が分析しやすいインサイトに変換する]

財務チームが財務情報を統合して、重要な業績[インサイト](finance-insights.md)をすばやく特定する方法について説明します。

>[!ENDTABS]
