# Acrobat人事ユースケースの概要

人事チームがAcrobatを使用して、従業員ライフサイクル全体の文書とワークフローを管理する方法について説明します。 複雑な人事資料の理解や関係者との共同作業から、ブランド上の明確なコンテンツの作成に至るまで、Acrobatを使用して人事チームがより効率的に作業し、一貫性を保ち、より良い社員体験を提供する方法をご覧ください。

## 新機能

>[!BEGINTABS]

>[!TAB インサイトからインパクトまで：オンボーディングワークフローの効率化]

人事チームがAcrobat Studioを使用して、従業員のオンボーディングコンテンツを[インサイト、コラボレーション、オンブランド配信](hr-insights.md)へと効率化する方法について説明します。

>[!ENDTABS]
