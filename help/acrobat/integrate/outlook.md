# 電子メールメッセージと添付ファイルのPDFへの変換 [!DNL Outlook]

電子メールメッセージと添付ファイルを [!DNL Outlook] プロジェクトに最適です。 添付ファイルを自動的に情報に変換することで、よりプロフェッショナルで安全な方法で情報を提供する方法についてPDFします。 このツールは Windows でのみ使用できます。
