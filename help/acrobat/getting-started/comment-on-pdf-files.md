# PDFに対するコメント

Webブラウザーのみを使用して、共有PDFにコメントを追加する方法について説明します。 チームとの共同作業でも文書の校正でも、Acrobatのコメントツールを使用すると、明確で実用的なフィードバックを簡単に提供できます。 このビデオチュートリアルでは、[新しいAcrobatエクスペリエンス](new-experience.md)を使用します。

PDFを共有する方法については、[リアルタイムで共同作業](collaborate.md)のチュートリアルをご覧ください。
