# レイアウトの自動調整

コンテンツをリフローし、レイアウトの複数ページにわたってレイアウトを自動調整する新しい編集モードについてPDFします。

>[!NOTE]
>
>レイアウトの自動調整では、まだ一部の種類の言語やPDFコンテンツはサポートされていません。
