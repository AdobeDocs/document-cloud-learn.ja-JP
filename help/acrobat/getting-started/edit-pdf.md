# PDF を編集

写真ファイルのテキストと画像を編集する方法についてPDFします。 クリックまたはタップして、新しい段落の追加、誤字の修正、またはPDFページの並べ替えを行います。 画像の追加や写真の切り抜きもできます。

>[!TIP]
>
>新しい編集モードについて学習します。 [コンテンツの自動調整](auto-adjust-layout.md)を選択します。
