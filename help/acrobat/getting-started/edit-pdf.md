# PDF を編集

使い方 [PDFファイル内のテキストや画像を編集](https://www.adobe.com/jp/acrobat/online/pdf-editor.html){target="_blank"}. クリックまたはタップして新しい段落を追加するか、誤字を修正します。 画像を追加することもできます [または写真を切り抜く](https://www.adobe.com/acrobat/online/crop-pdf.html).
