# ファイルを 1 つのPDFに

様々な種類のファイル (PDF、Word、Excel、PowerPoint、JPEGなど ) を 単一のPDFに 複数のファイルをひとつのPDFにまとめることで、他のユーザーに送信するコンテンツを管理および制御し、より優れたエクスペリエンスを実現できます。

>[!TIP]
>
>ブランドを追加する方法 [新しいカスタムページ](add-custom-page.md) ファイルの結合を行います。
