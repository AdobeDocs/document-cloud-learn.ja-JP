# ファイルを1つのPDFに結合する

さまざまな種類のファイル（PDF、Word、Excel、PowerPoint、JPEGなど）を組み合わせる方法を説明します。 を1つのPDFに変換します。 ファイルを1つのPDFに結合すると、他のユーザーに送信するコンテンツを管理および制御でき、より優れた操作性を実現できます。
