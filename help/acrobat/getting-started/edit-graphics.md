# PDF内のグラフィックの編集

Acrobatの編集モードを使って、デスクトップとwebのPDFー内で直接画像の追加、置換、切り抜き、サイズ変更、回転、配置をおこなう方法を説明します。 このチュートリアルでは、Adobe Expressを利用したアプリ内画像ジェネレーターを使用して、テキストプロンプトから画像を生成し、調整する方法についても説明します。 このビデオチュートリアルでは、[新しいAcrobatエクスペリエンス](new-experience.md)を使用します。

>[!VIDEO](https://video.tv.adobe.com/v/3431260?enablevpops&quality=12&learn=on&hidetitle=true)

[!BADGE チーム固有]

## 編集モードが様々なチームに役立つ仕組み

* **法務**

  法務チームは、編集モードを使用して、ファイルを再作成することなく、契約書、ポリシー、ガイダンスのグラフィックおよび視覚要素を更新し、レビューや規制の変更の際に、文書を正確かつ最新の状態に保つことができます。

* **Sales**

  セールスチームは、編集モードを使用して、提案や営業資料の画像の置き換え、レイアウトの調整、およびビジュアルの更新を行い、特定の顧客や商談に合わせてドキュメントをカスタマイズします。

* **時間**

  人事チームは、編集モードを使用して、政策文書、オンボーディング資料、トレーニングPDFの画像とレイアウトを変更するため、ソース設計ファイルに戻ることなく、迅速に更新を行うことができます。

* **財務**

  財務チームは、編集モードを使用して、財務PDFのチャートとビジュアルをすばやく更新し、ソース・ファイルに戻らずにレポートの正確性と正確性を維持できます。

* **調達**

  調達チームは、編集モードを使用して、元のファイルを必要とせずに、ベンダーの提案または契約のサプライヤーチャート、製品画像、および視覚的な詳細を迅速に更新することができます。
