# でのテキストタグの使用 [!DNL Microsoft Word]

Acrobat Signテキストタグを [!DNL Microsoft Word]を選択します。 文書を署名用に送信すると、テキストのタグ付けによって自動的にフォームフィールドが作成されます。フィールドを手動で配置する必要はありません。 このプロセスによって、 [!DNL Microsoft Word] 署名用に送信する文書。
