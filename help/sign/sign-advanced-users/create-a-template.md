# 文書テンプレートの作成

再利用可能な文書テンプレートを作成して、後で使用できるようにAcrobat Signライブラリに追加する方法について説明します。 文書テンプレートを使用すると、署名トランザクションを迅速かつ一貫性のある方法で処理できるほか、文書の改訂や変更を管理する機能も利用できます。
