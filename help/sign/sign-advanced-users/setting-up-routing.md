# 署名順序の設定

複数の署名者の署名順序を設定する方法について説明します。 文書を順番に送信したり、並行して送信したり、特定のグループに送信したりできます。

>[!NOTE]
>
>ハイブリッドルーティングは、エンタープライズレベルのプランでのみ使用できます。 Acrobat Sign管理者は、アカウント設定の「送信設定」セクションでこれらの機能を有効にする必要がある場合があります。
