# 署名順序の設定

複数の署名者の署名順序を設定する方法について説明します。 文書を順番に送信したり、並行して送信したり、特定のグループに送信したりできます。

>[!NOTE]
>
>ハイブリッドルーティングは、エンタープライズレベルのプランでのみ使用できます。 このチュートリアルで説明する特定の設定を有効にするには、Acrobat Sign管理者に連絡する必要があります。
