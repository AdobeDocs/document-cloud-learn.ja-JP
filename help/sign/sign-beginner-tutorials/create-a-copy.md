# 「コピーを作成」を使用する場合

既に契約書を送信しましたが、有効期限が切れているか、キャンセルされました。または、もう一度送信する必要がありますか？ コピーを作成すると、既存の契約書を複製し、変更が必要なものを調整して、数秒で再送信できます。 最初から再構築できません。 「コピーを作成」は、次の場合に使用できます。

* 契約書の元の送信者です
* 契約書は、最新の署名依頼エクスペリエンスから送信されました
* 契約書に証言受信者の役割が含まれていません
* 文書の保持またはGDPRの削除のため、契約書は削除されていません
* 契約書では、送信者がアクセスできなくなったライブラリテンプレートへのアクセスは必要ありません

>[!NOTE]
>
>「コピーを作成」は、エンタープライズレベルのプランで使用できます。
