# 送信オプションの設定

署名用に文書を送信する際に、契約書の詳細をカスタマイズしたり、ファイルを添付したり、リマインダー、有効期限、受信者の言語などの追加設定を構成したりする方法について説明します。

>[!NOTE]
>
>管理者が送信オプションをカスタマイズしている可能性があるため、表示されるオプションはこのチュートリアルで示されるオプションとは異なる場合があります。
