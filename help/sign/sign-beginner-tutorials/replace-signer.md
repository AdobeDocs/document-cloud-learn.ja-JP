# 署名者の置き換え

署名用に文書を送信する際に間違った電子メールを使用した場合など、署名者を置き換える方法について説明します。

>[!NOTE]
>
>「受信者を置き換え」のデフォルト設定では、現在の受信者が新しい受信者に完全に置き換えられます。 ビジネスレベルおよびエンタープライズレベルのプランには、追加のオプションがあります。
