# 概要

![「はじめに」に署名の画像](../assets/Hero-GettingStarted.png)

以下の簡単な手順を踏んだチュートリアルで、文書を送信、署名、トラックする方法を習得しましょう。 Acrobat Signのクイックツアーを参照した後、1 人または複数のユーザーへのドキュメントの送信に進みます。 このコンテンツは、電子サインのワークフローを容易に開始できるように設計されています。
