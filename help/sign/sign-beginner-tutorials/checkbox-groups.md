# チェックボックスグループを使用した必要な選択のガイド

チェックボックスグループを使用して、署名者がフォームで選択できるオプションの数を制御する方法について説明します。 チェックボックスグループを使用すると、契約書を送信する前に、最小または最大数の選択を要求するなどの簡単なルールを適用して、署名者が文書を正しく完了できるように支援できます。

チェックボックスグループの詳細：

* 提出物が不完全または無効にならないようにする
* 明確なインラインフィードバックで署名者をガイド
* フォームで収集される情報の精度を向上させる
