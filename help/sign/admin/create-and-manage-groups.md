# グループの作成と管理

グループの作成、グループへのユーザーの追加、およびグループ設定の編集方法について説明します。 グループ機能を使用すると、別のライブラリ文書や別の設定（ロゴや電子メールの CC に指定された関係者など）にアクセスするユーザーを、それぞれのグループに分けることができます。

>[!NOTE]
>
>グループは、ビジネスレベルおよびエンタープライズレベルのプランでのみ利用できます。
