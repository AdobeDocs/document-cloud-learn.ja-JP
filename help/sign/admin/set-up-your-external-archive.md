# 外部アーカイブの設定

署名された契約書のバックアップコピー用の外部アーカイブを作成する方法について説明します。 コピーは、任意の電子メールアドレスまたは [!DNL Box] とEvernoteです。 署名済みの完成した電子メールのコピーが、指定した電子メールアドレスまたはクラウドサービスに送信されます。
