# レポートチャートの作成

新しいレポートエクスペリエンスを使用してパーソナライズされたビューで独自のカスタムレポートとデータ書き出しを作成、保存、管理する方法について説明します。 レポートチャートが使用できるかどうかは、ユーザーの Acrobat Sign システムの権限レベルによって異なります。

>[!NOTE]
>
>レポートは、ビジネスレベルおよびエンタープライズレベルのプランでのみ使用できます。
