# 契約書を変更の設定と管理

Acrobat Signで&#x200B;**契約書を変更**&#x200B;を構成および管理する方法について説明します。 このチュートリアルでは、管理者がアカウントレベルおよびグループレベルのコントロールを設定し、適格性ルールを理解し、送信後に変更が行われたときに契約書の整合性がどのように保持されるかを学習します。 送信者のアクションを追跡および監査して、コンプライアンスと透明性をサポートする方法について説明します。

学習内容：

* 契約書の変更を有効または制限する管理者設定
* 契約書を変更できる場合と変更できない場合
* 改訂モードによって契約書の整合性がどのように保護されるか
* 監査レポートに記録される変更
