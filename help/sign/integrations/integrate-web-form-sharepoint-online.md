# Webフォームの統合 [!DNL SharePoint Online]

Webフォームから収集したデータを [!DNL SharePoint] リストで、チーム全体で収集したデータを共有できます。

>[!NOTE]
>
>Acrobat Sign Webフォームと [!DNL SharePoint] リストは、オンラインインストールでのみ使用でき、オンプレミスインストールでは使用できません。
