# Web フォームを [!DNL SharePoint Online]

Web フォームから収集したデータを [!DNL SharePoint] リストを表示し、チーム間で収集したデータを共有できます。

>[!NOTE]
>
>Acrobat Sign Web フォームと [!DNL SharePoint] リストはオンラインインストールでのみ使用でき、オンプレミスインストールでは使用できません。
