# Salesforce 用Acrobat Sign でのカスタムボタンの作成

Salesforce.com で標準オブジェクトのカスタムボタンを作成する方法を紹介します。この例では、カスタムボタンが **連絡先** を選択します。 ボタンをクリックすると、送信プロセスが開始され、 **連絡先** 契約書に直接追加できます。
