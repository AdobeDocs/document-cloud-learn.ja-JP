# 簡単なワークフローを作成するには [!DNL Power Automate]

組織の技術者以外のメンバーが、事前定義済みの [!DNL Microsoft Power Automate] Acrobat Signに無償で含まれているコネクタ。 これらの自動化された文書ワークフローにより、一般的なプロセスの速度と効率が向上し、繰り返し行う作業にかかる時間が短縮され、エラーや作業のやり直しが最小限に抑えられます。
