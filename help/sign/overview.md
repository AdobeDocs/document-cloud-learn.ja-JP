# Acrobat Signチュートリアル

Adobe Document CloudソリューションであるAcrobat Signは、統合、API、高度な認証、追加の管理機能などを使用して、組織のニーズに合わせて拡張できます。 ここでは、Acrobat Signを使って初心者と管理者の両方に学習のスピードを速めるために設計された、幅広い学習体験を見つけることができます。
