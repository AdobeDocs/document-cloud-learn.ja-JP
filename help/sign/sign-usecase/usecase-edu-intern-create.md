# 教育 Web フォームインターンシップ応募の作成

学生が必要な書類にスムーズにアクセスして任意のデバイスと場所から送信できる、インターンシップ用 web フォームの作成方法を説明します。 この例では、すべての学生に対するワークフローが、学習コースに応じて固有になっています。これは、各学生が承認を得るために、独自の教員スーパーバイザーと部門長を追加する必要があるためです。
