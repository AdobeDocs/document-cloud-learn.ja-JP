# Acrobat Signの Liquid Mode

Liquid Mode がモバイル署名エクスペリエンスをどのように改善するかご覧ください。 文書の読み取りや署名を試みる際に、ピンチズームする必要がなくなりました。 このチュートリアルでは、 _HTML型_ 署名用に送信することで、モバイルユーザー向けに強化された署名エクスペリエンスを自動的に作成できます。
