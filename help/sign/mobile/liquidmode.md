# Acrobat SignのLiquid Mode

Liquid Modeによりモバイル署名エクスペリエンスがどのように向上するかについて説明します。 文書の読み取りや署名を行うときに、ピンチやズームが発生しなくなりました。 このチュートリアルでは、 _HTMLフォーム_ 署名用に送信すると、モバイルユーザー向けの強化された署名エクスペリエンスを自動的に作成できます。
