# モバイルの概要

モバイルデバイスを使用して、署名用に文書を送信したり、電子サインの進捗状況やリアルタイムの更新情報を取得したりすることができます。

## 新機能

>[!BEGINTABS]

>[!TAB モバイル対応ビュー]

[モバイル対応ビュー](mobile-friendly.md)を使用して、モバイルデバイスでフォームに入力する方法を説明します。

>[!TAB モバイル対応ビューの作成]

開発者のサポートなしに、[モバイル対応](create-mobile-friendly.md)のドキュメントをシームレスに生成する方法について説明します。
