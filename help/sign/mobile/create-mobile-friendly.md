# モバイル対応ビューの作成

開発者のサポートなしに、モバイル対応のドキュメントをシームレスに生成する方法について説明します。

モバイルビューでは、送信者は次のメリットを得ることができます。

* 完了率の向上：フォームの完了率が高くなり、結果が改善される
* 時間とコストの削減：文書をモバイルフレンドリーなフォームに変換する手間を省きます
* 技術的な専門知識は不要：最小限の労力でダイナミックでモバイルフレンドリーなフォームを作成
