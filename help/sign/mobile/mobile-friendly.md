# モバイルフレンドリーな表示

外出先でタスクを完了するということは、もはや任意ではありません。契約書に署名する専門家から、学校のフォームに記入する親まで、シンプルなモバイルソリューションへの需要はかつてないほど高まっています。 モバイル対応ビューを使用して、モバイルデバイスでフォームを完了する際のピンチやズームを排除し、エラーを減らす方法について説明します。

Acrobat Signの「Mobile Focus」により、どこにいても、誰でもモバイルデバイスで簡単にフォームに入力して署名できます。 モバイルデバイスのUIを自動的に作成して簡素化すると、Mobile Focusはユーザーに必要なフォームフィールドを案内し、次の方法で文書を効率的かつ正確に完了できるようにします。

* モバイルフレンドリーなフィールド：画面に最適なフォームのピンチやズームが不要です
* 精度の向上：エラーを減らし、必要なすべての情報を確実に取得
* 継続性：ユーザーはいつでも、進捗状況を維持したまま、モバイルフレンドリーなフォームを開始して戻ることができます。
