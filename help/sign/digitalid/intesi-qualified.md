# デジタル ID の取得元 [!DNL Intesi Group] （認定）

適格な電子署名証明書を [!DNL Intesi Group]を選択します。 登録が完了し、身元が確認されたら、 [!DNL Intesi Group] Acrobat Sign クラウド署名の適用に使用するデジタル ID を発行します。
